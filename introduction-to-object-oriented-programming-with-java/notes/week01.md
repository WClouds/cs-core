## L1 Executing Java Programs
### Compilers
- compiler performs the translation of an entire program in one session and generates a new set of files, the low level code
- advantages: compiled language faster than interpreter language
- disadvantages: rely on the instruction sets of the processors, different processors have different sets, the programers would need to be compiled for every kind of instruction sets that exist
### Interpreters
- interpreter translates a program on the fly
- disadvantages: slow compared with compiled language
- advantages: do not require intermediate files to store particular kind of machine code, they offer greater platform independence. A programmer can write a program using an interpreted language and that same code will execute on any computer without requiring any extra work from the programmer.
### Java's Hybrid Approach
- compile program to bytecode, and interprete the low-level bytecode in real time
- high performance and platform independence
- Java Virtual Machine(JVM) run the compiled byte code
